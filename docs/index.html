Navidrome front end

NaviGlassPlayer

A browser-based glass-style player for Navidrome. Browse albums, artists, playlists, and genres with a WebGL carousel and built-in playback.

View on GitHub Run locally

Quick start

Clone the repo, start Navidrome, then run the local Node server:

git clone https://github.com/BASF2HD/NaviGlassPlayer.git
cd NaviGlassPlayer
npm start

Open http://127.0.0.1:8787 and sign in with your Navidrome username and password.

Subsonic API

Uses Navidrome's Subsonic-compatible REST API for library browsing, cover art, and streaming.

Local proxy

server.mjs serves the UI and proxies /navidrome/* to your Navidrome instance so the browser stays same-origin.

Deploy scripts

One-shot installers for Rocky Linux, Ubuntu, Synology DSM, and Raspberry Pi OS are included in the repo.

Smart playlists

Bundled .nsp templates for decade, language, and mood playlists live in smart-playlists/.

About this site

GitHub Pages hosts this project page only. The full player needs npm start (or your own reverse proxy) because Navidrome credentials are sent through the local proxy, not from static hosting.
